  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

Question-3: What is event delegation in the DOM?

Event delegation in the DOM (Document Object Model) is a technique where instead of attaching event listeners to individual child elements, you attach the event listener to a parent element and take advantage of the event bubbling mechanism to handle events on child elements
